# README_mgngtus_figures.md

R and MATLAB code for making the figures in the main text and the supplementary materials.

MGNG TUS STUDY, UNIVERSITY OF PLYMOUTH.
Copyright (C) Johannes Algermissen, University of Oxford, Oxford, UK, 2024-2025.

- *mgngtus_figure1.R*: Code for making figure 1F (peak pressure and temperature rise per participant per condition) in the main text.
- *mgngtus_figure2.m*: Code for making figures 2D-K line plots and bar plots of the proportion of Go responses and stay responses per task condition in the sham condition as well as model comparison results) in the main text. 
- *mgngtus_figure3.m*: Code for making figures 3A-H  (line plots and bar plots of the proportion of Go responses and stay responses per task condition in the TUS-dACC and TUS-aIns conditions) in the main text.
- *mgngtus_figure4.m*: Code for making figures 4B,C,E,F (bar plots of the parameter values per sonication condition) in the main text.
- *mgngtus_figureS3.R*: Code for making figures S3A-C (bar plots of mean RTs as a function of task condition and sonication condition) in the Supplementary Information.  
- *mgngtus_figureS4.m*: Code for making figures S4A-E (bar plots of the proportion of stay responses per per last outcome and last response for the sham condition, both empirically and simulated from M1, M3, M5, M7) in the Supplementary Information. 
- *mgngtus_figureS5.m*: Code for making figures S5A-E (bar plots of the proportion of stay responses per per cue valence for the sham condition, both empirically and simulated from M1, M3, M5, M7) in the Supplementary Information. 
- *mgngtus_figureS6.m*: Code for making figures S6A-K (line plots and bar plots of the proportion of Go responses and stay responses per task condition in the sham condition simulated from M7, M8, M9 as well as model comparison results) in the main text.
